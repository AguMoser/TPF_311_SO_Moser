## Descripción

Este repositorio contiene la resolución del trabajo práctico grupal e individual.

Cada integrante participó realizando commits propios simulando distintos roles del sistema:

- Sysadmin: creación de estructura America y estructura individual sysadmin.
- Manager: creación de estructura Europa.
- Dev: creación de estructura Asia.

Además se incluye un script grupal con los comandos utilizados para la creación de los continentes.
